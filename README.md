# Metal Detect

An open source Android application that turns your smartphone into a metal detector using the built-in magnetometer sensor.

## Description

Metal Detect uses your phone's built-in magnetometer sensor to detect metal objects. When metallic objects are near, the magnetic field intensity increases and the app provides visual, audio, and haptic feedback.

**Note:** This app is designed for entertainment purposes only and should not be used for serious metal detection activities.

## Installation

### From Source

1. Clone this repository:
   ```bash
   git clone https://github.com/MakeWake-ch/Metal-Detect.git
   ```

2. Open the project in Android Studio

3. Build and install on your device

### From F-Droid

Available on F-Droid once submitted.

## Features

- Real-time magnetic field strength display in μT (microTesla)
- Audio alert when metal is detected
- Haptic vibration feedback
- Light/Dark theme support
- Flashlight feature (on supported devices)
- Keep screen on option

## Requirements

- Android 7.0 (API 24) or higher
- Device with magnetometer sensor (most Android devices have one)
